Проект для удобных отчётов

ЗАПУСК
Запускается через index.py

ЭНДПОИНТЫ
• /register - POST
Принимает json с переменными "username", "password"
Возвращает строку с сообщением об успехе/ошибке
Создаёт нового юзера

• /login - POST
Принимает json с переменными "username", "password"
Возвращает json с токеном, который нужно записать в куки и убить через 24 часа, если при логине не выбрана галочка remember me
Записывает токен в БД, с которым будет сверяться при следующих операциях

• /add_api - POST
Принимает json с переменными "token", "username", "apiName", "apiKey"
Возвращает строку с сообщением об успехе/ошибке
Записывает новый апи ключ в аккаунт юзера

• /show_api - POST
Принимает json с переменными "token", "username"
Возвращает json со всеми апи ключами юзера

• /delete_api - POST
Принимает json с переменными "token", "username", "apiName"
Возвращает строку с сообщением об успехе/ошибке
Удаляет апи указанный апи ключ из юзера

• /sub_report - POST
Принимает json с переменными "token", "username", "dateFrom", "dateTo"
Возвращает json с отчётом по Афф Сабам

• /geo_report - POST
Принимает json с переменными "token", "username", "dateFrom", "dateTo"
